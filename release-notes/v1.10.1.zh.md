# 新功能

- 新增浏览器权限中心，集中查看并引导开启运行所需权限。
- OpenCLI Browser Bridge 支持自动拉起与自动连通修复，减少首次使用配置成本。
- 浏览器工具新增“仅重置数据”与“卸载”操作，支持清理 profile 与桥接相关配置。

# 改进与修复

- 修复 OpenCLI 扩展在部分场景下被 Chrome 判定为未启用的问题，并加入自动补救流程。
- 修复浏览器启动链路稳定性问题，避免“打开后立即退出”。
- 优化 Browser Bridge 安装与预热流程，提升安装后首次可用率。
- 修复发布流程顺序问题，并补充 platform 版本同步步骤。
- 统一运行时安装中的 Xcode 工具链自修复逻辑。
- 修复 Hermes 设置页标签显示问题，并新增运行时升级检查入口。
