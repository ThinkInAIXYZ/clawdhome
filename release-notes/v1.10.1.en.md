# New Features

- Added a browser permission center to check required permissions and guide users to enable them.
- OpenCLI Browser Bridge now supports automatic launch and connection remediation for smoother first-time setup.
- Added browser tool actions for data-only reset and uninstall, including profile and bridge cleanup.

# Improvements & Fixes

- Fixed cases where the OpenCLI extension was marked disabled by Chrome, with automatic remediation.
- Fixed browser startup stability issues that could cause immediate exit after launch.
- Improved Browser Bridge install and warm-up flow to increase first-run success rate.
- Corrected release pipeline ordering and integrated platform release seeding.
- Unified Xcode toolchain self-repair logic during runtime installation.
- Fixed Hermes settings tab display issues and added a runtime upgrade check entry.
